github+hexo构建自己的博客

  • other

宽敞漂亮的房子是每个人的追求，而对于我们的程序员来说，还需要一个自己的个人空间，那里记录了自己的汗水，记录着折磨过几天的天坑，是我们吃一堑长一智的见证，所有我们要追求一个高大上的地方，来存放自己的一片天地。

接下来我们就来建造一个属于自己的房子：github+hexo

那么我们建造的房子到底如何呢,我们先睹为快吧~

首先我们要搭建这样的个人博客需要2个材料要准备

Github

对于一个码农来说，没有一个属于自己的github账号那就太不怀兴了.

  1. 如果还没有的就赶紧去注册个吧。
  2. 我们已经有账号了就去看看github为我们准备的GitHub Pages吧，按照指引我们就可以完成地基的搭建

注意事项：

  1. 创建的repository name 必须和你的用户名一致为username.github.io后缀也不能改，如果想不用自己的用户名可以新创建organization，在下面创建repository username为organization的名称。
  2. 根据指引在clone 项目到本地后创建一个html界面，如果访问地址成功了，那个改步骤结束

Hexo

既然我们的房子已经建成，那么我们就要给它添砖加瓦，变得更漂亮，ok！

hexo的安装

首先请出hexo官方文档
我们可以在这里进行hexo的安装，安装完hexo后你就应该具备git,node环境了，并且根据上个步骤的操作，你的本地已经clone了一份GitHub上的代码了。

下面介绍如何新建一篇文章：

  1. 在我们的项目地址上执行
  2. $ hexo new [layout]
  3. hexo g
  4. hexo d
